# SociActive Frontend

A complete React application for outdoor social activities.

## Structure
- `src/api`: Axios clients split by resource (Activity and User).
- `src/components`: Reusable UI elements.
- `src/context`: Global state (Auth).
- `src/hooks`: Custom React hooks for data fetching and logic.
- `src/pages`: Screen components.
- `src/router`: Routing definitions and protection.
- `src/styles`: CSS variables and global styles.
- `src/utils`: Helpers, constants, formatters, and validators.

## Setup
1. `npm install`
2. Create `.env` from `.env.example`
3. `npm start`
